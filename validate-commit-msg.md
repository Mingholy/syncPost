# Semantic Commit Message
```
<type>(<scope>): <subject>
```

## types
* `feat`
* `fix`
* `docs`
* `style`
* `perf`: optimizationg on performance
* `refactor`
* `test`
* `chore`: changes that don't touch code and test case

## scopes
* `init`
* `runner`
* `watcher`
* `config`
* `web-server`
* `proxy`
